return使用注意点

Oct 18, 2018 | Javascript | 阅读

场景描述

一天晚上，我在控制台输出了这样一段代码，结果给我报了个非法的返回语句错误？咿，for循环中不能用return吗？以前也用过呢！这次是咋回事，一时难以理解。

为什么呢

return语句终止函数的执行，并返回一个指定的值给函数调用者。

MDN -return

查了return的定义，才注意到之前用一直在函数里面，而这次控制台直接输出并没有在函数里使用，所以报错了！return 应该运行在函数中才可以的！
